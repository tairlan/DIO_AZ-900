Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO.

Resumo – Primeira aula AZ-900

Na primeira aula da certificação AZ-900, aprendi os conceitos iniciais de computação em nuvem. Foi explicado o que é a nuvem, como ela funciona e por que as empresas utilizam esse modelo.

Vimos também os principais benefícios da computação em nuvem, como:

Redução de custos

Escalabilidade

Flexibilidade

Alta disponibilidade

Pagamento conforme o uso

Foi abordado o conceito de custos na nuvem, explicando que o cliente paga apenas pelos recursos que utiliza, evitando gastos com infraestrutura física.

A aula apresentou os modelos de serviço:

IaaS (Infrastructure as a Service)

PaaS (Platform as a Service)

SaaS (Software as a Service)

Também foi explicado, de forma básica, o SLA (Service Level Agreement), que define os níveis de disponibilidade e garantia dos serviços oferecidos pela Microsoft Azure.

Por fim, foi mostrada a interface do Azure Portal, apresentando o básico do ambiente, como:

Visão geral do portal

Alteração de idioma

Opções de personalização

Essa aula serviu como uma introdução aos conceitos fundamentais da nuvem e ao ambiente do Azure.
